Overview

I worked on this project as a mentor for the UNLV Girls in Ubiquitous Intelligence (GUIC) summer camp. This is a program that seeks young women (grades 6-12) interested in STEM to work on an Arduino project over the summer and learn more about programming. My group was tasked with creating an attendance checking system to efficiently take attendance in a smart classroom.
